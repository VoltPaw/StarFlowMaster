# StarFlow Master

## 简介
**StarFlow Master** 是一款专为天文摄影爱好者打造的一站式极客星轨合成引擎。**特别面向新手设计，无需繁琐的 Photoshop (PS) 堆栈或 Premiere Pro (PR) 视频剪辑，即可轻松完成星轨合成与延时视频制作，彻底告别后期烦恼。**

它基于 Python 和 PyQt5 开发，**采用极具科技感的深空毛玻璃 UI 设计**，界面美观精致，交互流畅自然。不仅提供了高效、精准的星轨合成、视频生成以及图像堆栈功能，更让每一次创作都成为一种视觉享受。无论你是想要合成一张震撼的星轨大片，还是制作一段流动的星空延时视频，StarFlow Master 都能助你一臂之力。

## 核心功能

*   **1. 最大值堆栈 (Global Max-Hold Stacking)**
    *   **常规星轨堆栈**：采用经典的最大值叠加算法（Max-Hold），将一系列连续拍摄的星空照片合成一张完整的星轨大片。
    *   **高效稳定**：针对海量素材优化，利用多线程并行处理技术，大幅缩短合成时间；同时优化内存管理，即使在普通配置的电脑上也能流畅处理高分辨率素材。

*   **2. 残影延时视频 (Afterimage Timelapse Video)**
    *   **动感视觉**：不仅能生成静态图片，还能生成带有“拖尾残影”效果的动态延时视频。
    *   **流光溢彩**：通过PR同款残影算法，让每一颗星星都拖着长长的轨迹划过夜空，营造出流星雨般的梦幻视觉体验，比传统延时视频更具冲击力。支持自定义帧率与画质，直接输出高品质视频文件。

*   **3. 最大值星点堆栈 (Star-Only Max-Hold Stacking)**
    *   **解决痛点**：为了解决传统最大值堆栈容易导致地景边缘模糊、以及天地合成时边缘存在明显割裂感（黑边/白边）这两大难题，我们创新设计了“最大值星点堆栈”模式。
    *   **完美融合**：该模式能够将提取出的星轨完美合成到任意一张指定的底图中，实现星轨与地景的无缝融合。
    *   **核心原理**：内置专业的 **SEP (Source Extraction and Photometry)** 算法，精准检测每一帧画面中的星点位置，仅在星点出现的区域进行像素叠加，而保持地景和天空背景不动。
    *   **性能卓越**：精心的工程优化使得星点检测速度极快，且同样支持多线程加速与低内存占用模式，让复杂的蒙版合成变得简单自动化。


## 使用说明

### 第一步：导入素材
支持**拖拽上传**与**按钮添加**两种方式。直接将图片或文件夹拖入软件，或点击“添加文件/文件夹”按钮即可。
*   **注意**：请确保所有导入的图片**尺寸一致**。
*   **建议**：推荐使用 **JPG** 格式以获得最佳的处理速度与兼容性。

### 第二步：选择功能模式
软件右侧提供了三个功能标签页，点击即可切换：
*   **最大值全局堆栈**：适合制作传统的静态星轨图。
*   **残影延时视频**：适合制作流动的星空视频。
*   **最大值星点堆栈**：改良版星轨堆栈，强烈推荐。

### 第三步：设定参数与运行
根据选择的功能，设置相应的参数。以下是详细的参数含义与建议范围：

#### 1. 最大值全局堆栈
| 参数名称 | 含义 | 建议设置 |
| :--- | :--- | :--- |
| **输出文件名** | 生成图片的名称 | 默认即可，支持中文 |
| **并发线程数** | 同时处理图片的线程数量 | 建议设为 CPU 核心数 (如 8-16)，过高可能卡顿 |
| **缓存图片数** | 预读取到内存的图片数量 | 内存大（16G+）可设 32-64，内存小建议 8-16 |

#### 2. 残影延时视频 (推荐)
| 参数名称 | 含义 | 建议范围 / 说明 |
| :--- | :--- | :--- |
| **拖尾衰减系数** | 决定流星拖尾的长度。数值越大，拖尾越长。 | **0.90 - 0.99**<br>0.95: 适中拖尾<br>0.99: 极长拖尾（类似全星轨）<br>0.50: 短促流星 |
| **视频帧率 (FPS)** | 每秒播放的画面数量 | **24, 25, 30** 或 60 |
| **视频清晰度 (CRF)** | 数值越**小**画质越好，文件越大 | **18 - 23** (推荐 20) |
| **编码速度** | 编码越慢，压缩率越高（文件越小） | 一般选"快速"或"中等" |

#### 3. 最大值星点堆栈 (进阶)
| 参数名称 | 含义 | 建议范围 / 说明 |
| :--- | :--- | :--- |
| **星点检测范围** | (可选) 黑白蒙版图，白色区域检测，黑色屏蔽 | **无需 PS**！上传一张黑白照片，将需要星点检测的地方涂黑，不需要星点检测的地方涂白。若不填则检测全图星点检测。 |
| **星点亮度阈值** | 判定“多亮算星星”的标准 | **5.0 - 50.0**<br>数值越小，检测到的星星越多（可能误检噪点）；数值越大，只检测亮星。 |
| **星点最小面积** | 判定星星的最小像素点数 | **3 - 15**<br>用于排除单像素热噪点。 |
| **非检测膨胀** | 用于消除检测边界误检测的星点 | **0.5% - 2.0%** (推荐 1.0%) |
| **非星区域底图** | 用于填充非星点区域的图片 | 推荐选“**最后一张**”或“**自定义**。 |

*   **测试功能**：建议先点击“**测试星点检测**”按钮，选一张图看看红色圆圈是否准确圈住了星星。如果漏选，降低非检测膨胀值；如果误选地景，提高非检测膨胀值。
